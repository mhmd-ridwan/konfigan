  • Apakah aplikasi yang Anda mod itu aman?

    Tentu, sudah saya pastikan dahulu sebelum share. Namun terkadang ada device yang cukup sensitif masalah keamanan, seperti Xiaomi, abaikan saja jika itu terdeteksi tidak aman. Selama bukan terdeteksi sebagai Virus Trojan atau Virus lain, pasti itu aman terkendali.

  • Jika masih ada app original yang masih terpasang, apakah itu harus di uninstall terlebih dahulu?

    Ya, aplikasi original dan aplikasi mod itu sudah berbeda signature, jadi tidak akan bisa install versi mod jika versi original masih terpasang. Kecuali kalau mungkin salah satunya dibuat menjadi versi clone, maka kamu akan bisa install secara bersamaan.
